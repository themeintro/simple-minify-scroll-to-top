= Usage =

* Go to your Dashboard after installation and navigate to "Dashboard > Scroll To Top Option" to configure the accordion.

== Installation ==

1. Upload `simpleminifyscrolltotop` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
